# EventHive - Event Ticket Booking System

A responsive front-end website for browsing and booking tickets to movies, sports matches, and entertainment events. Built with plain HTML, CSS, and JavaScript.

## Features

- Semantic page layout with header, navigation, event listing, booking, and footer sections
- Event cards showing name, date, location, and an icon for each event category
- Toggleable event details panel (venue, time, ticket price)
- Booking form with name, email, event selection, and ticket count fields
- Front-end form validation with inline error messages
- "Book Ticket" buttons with click feedback
- Fully responsive layout for mobile, tablet, and desktop screens

## Getting Started

No build step or dependencies are required.

1. Clone or download this repository.
2. Open `index.html` directly in any modern web browser.

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, custom properties, media queries)
- Vanilla JavaScript (DOM manipulation, form validation)
